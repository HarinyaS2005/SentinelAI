# SentinelAI

SentinelAI is an AI-powered banking cybersecurity platform that detects insider threats and correlates cybersecurity telemetry with banking transactions.

The platform continuously monitors privileged user activities, login behaviour, system access logs, and transaction events. Machine learning models analyze behavioural anomalies and generate real-time risk scores for security analysts.

The solution helps financial institutions proactively identify insider misuse, reduce fraud, and strengthen cyber resilience through intelligent monitoring and explainable AI.
